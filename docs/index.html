Autonomous, self-improving short-form video factory

Overview

MoneyPrinterTurbo turns a topic into a finished vertical Short: an LLM writes the script & search terms, TTS produces narration, subtitles are generated, matching stock footage is downloaded, and MoviePy composites it all into an HD video. On top of that core pipeline sits an autopilot that runs entirely on free GitHub Actions runners — it picks trending topics, generates several script candidates, lets an LLM judge pick the best, renders only the winner, uploads it to YouTube, and learns from each run.

The system is split into two scheduled halves: a cheap daily job that refreshes the topic list from Google Trends, and an every-4-hours job that does the heavy render/judge/upload. They share state through tracked files under autopilot/, committed back to main between runs.
Scheduling: the cadence is driven by a Cloudflare Worker (cloudflare-cron/), not GitHub's native schedule: event — the latter best-effort drops scheduled runs (worst for new repos / under load), which kept the autopilot from firing. The Worker runs on Cloudflare's reliable cron and triggers each workflow via the workflow_dispatch REST API; the workflows keep only their workflow_dispatch: trigger (no schedule:, so they never double-fire and waste YouTube quota).

Components

services/task.py

Pipeline orchestrator. Runs script → terms → audio → subtitle → materials → video → cross-post; updates progress via the state store.

services/autopilot.py

Self-improving loop: trends → topics, candidates → judge → render winner → upload → learn. Holds the daily refresh_topics() too.

services/llm.py

Script, search-term, judge & metadata generation across ~20 providers (default Gemini here). Strips <think> blocks.

services/voice.py

TTS. Default free Edge TTS (also yields subtitle timestamps); Azure Speech & SiliconFlow optional.

services/subtitle.py

Subtitles via edge (reuses TTS timing) or local whisper. Edge falls back to whisper.

services/material.py

Downloads matching stock footage from Pexels/Pixabay, or uses local materials.

services/video.py

MoviePy 2.x compositing; subtitles rendered via Pillow (no ImageMagick).

services/youtube_upload.py

Direct YouTube Data API v3 resumable upload. OAuth refresh-token → access-token per run.

services/state.py

Per-task status/progress store: in-memory or Redis (toggled by enable_redis).

controllers/manager/

Task queue / concurrency: threads up to max_concurrent_tasks; memory- or Redis-backed.

cloudflare-cron/

Cloudflare Worker on a reliable cron that fires both GitHub workflows via the workflow_dispatch API — replaces GitHub's flaky native schedule:.
